¿Qué es BookStack?

Una plataforma de base de conocimiento autoalojada, de código abierto, construida con PHP y Laravel.

📖

Organización jerárquica

El contenido se organiza en Libros → Capítulos → Páginas, facilitando la gestión de documentación técnica a cualquier escala.

🔐

Control de acceso por roles

Sistema RBAC granular con roles de administrador, editor y visualizador. Permisos aplicables a nivel de libro, capítulo y página.

🔍

Búsqueda avanzada

Indexación de contenido completa con soporte para búsqueda de texto libre, filtros por tipo y etiquetas personalizadas.

📝

Editor WYSIWYG + Markdown

Dos modos de edición: editor visual enriquecido y editor Markdown, con soporte para diagramas, código con resaltado y tablas.

📤

Exportación múltiple

Las páginas se pueden exportar en formatos PDF, HTML, texto plano y Markdown, facilitando la portabilidad del contenido.

🔌

API REST & Webhooks

API REST completa para integración con sistemas externos. Soporte para webhooks, LDAP, SAML y autenticación OAuth2.

Estrategia de pruebas

Proceso progresivo de evaluación técnica a lo largo de los 4 sprints del proyecto.

🧪

Pruebas Unitarias Sprint 1

Validación de servicios individuales: BookService, PageService, PermissionService, SlugService y más, usando PHPUnit con mocks de dependencias.

🔗

Pruebas de Integración Sprint 2

Verificación de la comunicación entre módulos: frontend–backend, ORM–base de datos, API REST. Integración con CI/CD mediante GitHub Actions.

🌐

Pruebas de Sistema Sprint 3–4

Evaluación de flujos completos de usuario: creación de libros, gestión de permisos, búsqueda, exportación. Pruebas de aceptación y despliegue automatizado.
